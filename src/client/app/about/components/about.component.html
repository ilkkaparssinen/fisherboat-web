Features of the boat

  • Raspberry Pi
  • Two AC motors, skid steering
  • GPS
  • On board camera
  • Fishing rig with flex sensor to detect catches
  • Python client connected via websocket to server
  • Speakers
  • 4G connection to internet

Server features

  • Node.js server for web sockets
  • Server can serve multiple boats
  • Simple pub/sub system for passing information between boat and client(s)
  • Serves the web ui html and javascript files
  • Amazon EC2

Web client features

  • Connects to server via web sockets
  • Angular 2
  • Map and simple MJPEG video
  • Controls the boat
